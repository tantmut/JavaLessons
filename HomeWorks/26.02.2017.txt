1) Создать и вывести таблицу умножения с помощью двумерного массива.
2) Подсичать сколько раз в строке "Hello world" встречается символ 'o', 'l'. Использовать методы String
3) Поменять в строке "Hello world" подстроку "Hello" на "GoodBye". Использовать методы String

4) Создать имитацию коллекции ArrayList для работы с типами int на основе массива.
Класс должен выполнять следующие операции:

   1) добавление элементов, учитывать, что массив может быть уже заполнен.
   2) изменение/удаление элементов по индексу.
   3) увеличение коллекции на заданное количество элементов.
   4) уменьшение коллекции до заданного количество элементов.
   5) вывод элементов в консоль в прямом и обратном порядке.
   6) сортировка листа(пузырьковая сортировка).

   При удалении элемента не обнулять его, а удалять физически.
